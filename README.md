# 🧠 Neural Screening System
**Hackathon Champion - REVA Hackathon 2026**

Advanced AI-powered cognitive risk screening using speech biomarkers, linguistic analysis, memory tests, and pronunciation assessment.

## ✨ Features
- 🎤 Real-time speech hesitation detection
- ✍️ Linguistic complexity analysis  
- 🧠 Multimodal memory recall matrix
- 🗣️ Articulation precision testing
- 📋 Professional PDF clinical reports

## 🚀 Quick Demo (30 seconds)
1. Click **START RECORDING** 
2. Say: `"I went shopping today"` → **15% LOW RISK**
3. Say: `"Uhh... umm... shopping..."` → **75% HIGH RISK**
